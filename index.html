What's Assist mode?
Assist mode is an optional feature which makes tracking all bonus stars a lot easier. Only select the numbers gotten, items used, space landed on etc and it automatically calculates the result and adds it to the counters.
It is recommended to use Settings Popout (bottom left) for this.
When switching from popout back to normal settings it might not display the correct state, just load the save from inside the assist settings (top left) when that happens.

Report Bugs
In case something breaks or doesn't provide the expected output, please click on "Generate & Copy" and contact me or open a Github issue with a screenshot or copy-paste of your browsers console (Ctrl + Shift + I > "Console"). It shows the last actions done and potential errors. Thank you!

Potential broken features
Mario Party is hugely RNG based so some events are rare and as such hard to figure out how they work.
Be sure to check your coin count after some of these as it might not be correct anymore:
  • SMP tied Minigame rewards
  • Battle Minigames (especially ties)
  • Lucky/ Extra- Bad Luck Events (some might even be missing)
  • VS space in SMP
  • MPDS Duel space

Welcome to Mario Party Overlay (MPO)! MPO helps track all kinds of stats and bonus stars from the entire Mario Party series, supporting every game in it!
This quick tutorial should tell you how everything works with some tips that might come in handy!

You can access this dialog at any point by clicking on the gear at the top and then selecting "Credits". The navbar can only be used once this dialog is closed.

Once settings are closed you can select your game at the top (it's the fifth entry in the navbar). Then select your characters for all four players by clicking on the character entries. Counters can be changed in the sixth entry in the navbar.

To start tracking, just click on the counters/numbers to increase them.
Coin stars can be tracked by clicking on a character. To avoid ties you can hold Ctrl to only give that character a coin star. Click on the coin star itself to change the amount.

To make tracking easier you can hold Ctrl to subtract them and hold Shift to increase/decrease by 5 instead of one. Both of them can also be manually changed in the navbar.
By clicking on the gear you can access some more options like saving all stats which you should probably do every now and then! If you ever accidentally reset all stats, you can still reload them there.

The "Generate & Copy" button next to it allows you to output all stats as text, this is useful in case you track stats for a streamer so you can post them in the chat. Don't forget to customize it under the General tab with advanced options shown!

Alternatively, you can also select the characters and counters you want in the Characters and Counters tab in here.
In Savefiles you can use various savefiles to track multiple games at the same time or backup everything locally as a file.

Assist mode makes tracking a lot easier as you only have to select the numbers rolled, items used, space landed on etc; MPO then automatically adds everything to the counters accordingly. Note that Assist mode is still in Beta.
